# ITB Map

## Deskripsi

Aplikasi ini adalah aplikasi yang dapat menampilkan peta, mengirim dan menerima request dari dan ke server untuk mendapatkan longitude dan latitude suatu lokasi dan menampilkannya kepada peta.
